Adding Colaberry Logo
1.    From Power BI Desktop ribbon click Insert and click Image.
2.    Select Colaberry logo from file.
3.    Under Visualization pane, click General and fill in these figures.
Size: Height = 83, Width = 96
Position: Horizontal =0, Vertical =0




 Adding Project Title
 
1.    Click on the Text Box from Home ribbon.
2.    Type in the Project Title
3.    Select Font Type Segoe Bold, Italic, and Font size 24, and 
4.    Alignment: Center the Title
5.    Format: Click on General and under properties, Size: Height =80, Width = 976, Position: Horizontal = 144, Vertical =0
